## Description

This repo is for Stamps Mini Test

## Installation

### Prerequisites

- Python 3.x

### Install Required Packages

```bash
pip install requests python-dotenv
```

## Setup

1. Create a `.env` file in the project directory with your OpenWeatherMap API credentials:

```env
OPENWEATHER_API_KEY=your_api_key_here
BASE_OPENWEATHER_URL_API=https://pro.openweathermap.org/data/2.5/forecast/
```

2. Get your API key from [OpenWeatherMap](https://openweathermap.org/api)

## Usage

Run the application, both app share the same syntax to run:

```bash
python main.py
```
